### Include Header
#### Yaitu memasukkan sebuah file header external ke c++

### Cara Compile dan membuka
1. Buka Terminal (Tekan Windows + R) lalu ketik cmd
2. Masuk Ke Directory
3. ketik
```
g++ main.cpp
```
lalu enter

4. Ketik : a
5. Lalu Enter
